# 🦉 OWL Hunt Game

A modern, interactive web-based hunting game where players test their reflexes by clicking on appearing owls before time runs out.

## 🎮 Game Features

### Core Gameplay
- **Timed Challenge**: 30-second rounds to score maximum points
- **Multiple Difficulty Levels**: Easy, Medium, and Hard modes with varying owl speeds
- **Scoring System**: 
  - Small owls: 1 point each
  - Large owls: 5 points each
- **Dynamic Owl Spawning**: Randomized appearance patterns across 36 grid positions
- **Real-time Score Tracking**: Live score updates during gameplay

### User Interface
- **Professional Game Header**: Displays real-time score and countdown timer
- **Animated Elements**: Smooth transitions, hover effects, and visual feedback
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Custom Modal System**: Beautiful game-over screen with performance feedback
- **Modern Visual Design**: Gradient backgrounds, glass-morphism effects, and custom animations

### Data Management
- **Leaderboard System**: Top 5 high scores tracking
- **Player Profiles**: Name-based score tracking
- **Persistent Storage**: Backend API integration for score persistence
- **Performance Feedback**:Custom motivational messages based on final score

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and modern structure

- **Bootstrap 5**: Responsive grid system and components

### Backend Integration
- **RESTful API**: Node.js/Express backend (assumed)

## 🎯 How to Play

1. **Enter Your Name**: Type your hunter name in the input field
2. **Select Difficulty**: Choose Easy, Medium, or Hard mode
3. **Start Game**: Click "Start Game" to begin the challenge
4. **Hunt Owls**: Click on appearing owls before they disappear
5. **Maximize Score**: Target large owls (5 points) over small owls (1 point)
6. **Beat the Clock**: Score as many points as possible in 30 seconds
7. **Check Leaderboard**: See if you made the Top 5!
